📝 简介

Claude Code 的实时状态栏 HUD，能显示上下文使用量（Token 条）、当前调用的工具（Read/Edit/Grep 次数）、子智能体运行状态、Todo 进度。直接在终端底部渲染，无需 tmux 分屏。对程序员：编程时清楚知道 Claude 消耗了多少上下文、在干什么。内容角度：Claude Code 用户极多，"可视化调试 AI 行为"是个有趣的角度。
